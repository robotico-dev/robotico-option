Reference **Robotico.Option** when you use the **Option pattern** (Option/Maybe). Type `Option<T>` (struct) with None, Some, IsSome, IsNone, Match, Map, Bind, GetValueOr(default).

## Installation

```bash
dotnet add package Robotico.Option
```
